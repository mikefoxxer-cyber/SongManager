# SongMaster - Minimal Jetpack Compose + Room Starter

This project is a minimal starter Android app with:
- Jetpack Compose UI
- Room local database for songs
- Simple editor and list UI

## Open in Android Studio
1. Open Android Studio.
2. Choose 'Open' and select the project folder.
3. Let Gradle sync (you may need to install required SDKs and plugins).
4. Build > Build Bundle(s) / APK(s) > Build APK(s)

APK will be created in `app/build/outputs/apk/debug/app-debug.apk`.
